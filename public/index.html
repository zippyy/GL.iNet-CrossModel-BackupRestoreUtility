What transfers between models

The utility deliberately avoids raw flash-clone behavior.

Transfers after validation • Wi-Fi profiles, subject to target band support • VPN profile data when matching target configuration exists • Named firewall entries and port-forward rules • AdGuard Home and DDNS only when their target configuration exists • Package manifest entries resolved from target feeds • Custom scripts/files safely staged by default • ELF binaries only where source and target CPU architectures match Requires manual review • WAN protocol and upstream credentials • LAN addresses, DHCP scope, VLANs, bridges, DSA/switch layout, and physical port mapping • Radio paths, country limits, channel width, and 6 GHz capability Never copied • Firmware, kernel, device identity, accounts, passwords, SSH host keys, and raw hardware-specific settings
